# Oman Next-Generation Sequencing & Bioinformatics Training

## Course overview
This course will provide training in laboratory and bioinformatics protocols for pathogen genomics, with a focus on SARS-CoV-2.

## Instructors
- Darrell Dinwiddie
- Daryl Domman
